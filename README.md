# Single-cell Sequencing on Human Fibrous Dysplasia of the Bone  

This repository contains scripts associated with the manuscript:  
[Single-cell analysis of human fibrous dysplasia bone reveals a fibrotic transcriptome and somatic, mosaic GNASR201H/C mutations in endothelial, perivascular, and stromal cells](https://www.cell.com/ajhg/fulltext/S0002-9297(25)00288-5) 

The project focuses on computational analysis of scRNA sequencing data from human FD and non-FD bone lesions.  

Raw data files are availible online through [NCBI's GEO database](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE301458).
